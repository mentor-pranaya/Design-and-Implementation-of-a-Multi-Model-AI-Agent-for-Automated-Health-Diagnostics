                          ❓ QUESTIONS & ANSWERS
═══════════════════════════════════════════════════════════════════════════════════

Q: Why can't my site be reached?
A: Python (the programming language) is not installed. Your entire application needs 
   Python to run. Install it from https://www.python.org/downloads/

Q: Why do I need to restart after installing Python?
A: Python needs to be added to your system PATH (where Windows looks for programs).
   PATH changes only take effect after a restart.

Q: Will I lose my data if I restart?
A: No! Restarting won't affect your data. It's safe and necessary for Python to work.

Q: How long does Python installation take?
A: 2-3 minutes for download and installation, then 1-2 minutes for restart. Total: ~5 min.

Q: Can I use Python 3.10 or 3.12 instead of 3.11?
A: Yes! Any Python 3.8 or newer works. 3.11 is just recommended.

Q: What if setup_and_start.bat doesn't work?
A: Run diagnose.bat to see what's wrong. It will tell you exactly what needs fixing.

Q: Can I close the terminal window while the server is running?
A: No! The terminal window is how the server runs. Keep it open while using the website.
   To stop the server, press Ctrl+C in the terminal.

Q: What's the "port 10000" thing?
A: It's where the server listens for requests. Like a mailbox address for your website.
   Standard ports are 80 (HTTP) and 443 (HTTPS), but 10000 works fine for local use.
