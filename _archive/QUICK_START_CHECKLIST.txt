==================================================================
                    TROUBLESHOOTING
==================================================================

❓ "Python not found" error?
  → You didn't restart computer after install
  → Or "Add Python to PATH" wasn't checked
  → Fix: Restart, or reinstall Python with checkbox checked

❓ "Address already in use"?
  → Server already running
  → Or port 10000 in use
  → Fix: Kill other processes, or use different port

❓ Server won't start?
  → Run: diagnose.bat
  → It will tell you what's wrong

❓ Still can't reach website?
  → Wait 30 seconds - server may be starting
  → Check in Windows Firewall - allow Python
  → Try: http://127.0.0.1:10000 instead
