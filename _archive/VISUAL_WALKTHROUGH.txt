                    TROUBLESHOOTING
════════════════════════════════════════════════════════════════

❌ ERROR: "Python not found"
   In terminal:
       'python' is not recognized...

   Fix:
       → You didn't restart computer after installing Python
       → Or "Add Python to PATH" wasn't checked
       → Solution: Restart computer, try again


❌ ERROR: "Address already in use"
   In terminal:
       Address already in use...
       
   Means:
       → Port 10000 already has a server on it
       → Or setup script ran multiple times
       
   Fix:
       → Close any other terminal windows
       → Kill any Python processes in Task Manager
       → Try again


❌ ERROR: "No module named 'fastapi'"
   In terminal:
       ModuleNotFoundError: No module named 'fastapi'
       
   Means:
       → Dependencies weren't installed properly
       
   Fix:
       → Make sure setup_and_start.bat completed fully
       → Try running it again
       → Run: diagnose.bat


❌ BROWSER: "Cannot reach localhost"
   In browser address bar:
       refused to connect
       
   Means:
       → Server not running
       → Or browser using wrong address
       
   Fix:
       → Check terminal window - see "Uvicorn running" message?
       → If yes, wait 5-10 more seconds
       → If no, restart: double-click setup_and_start.bat again
       → Try address: http://127.0.0.1:10000
